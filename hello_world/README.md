A Flutter learning project with an animated Hello World screen.

[View Assignment PDF](Assigment_assets/Assignment_flutter_merged.pdf)

### What I Did

- Implemented Provider state management with a counter example demonstrating ChangeNotifier pattern
- Created CounterModel class that extends ChangeNotifier to manage state and notify listeners on changes
- Used Consumer widget for efficient UI rebuilds and context.read() for state updates without listening
